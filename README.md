# delight 阿贝水果店项目

> NodeJs + VueJs (Express + ElementUI)

>本项目为模拟水果商城，是一个完整前后端分离的项目

>本项目分为前端，后台管理和服务器

## 服务器

>作为数据库提供数据，端口设置为3000，使用MongoDB

>优先部署

## 后台管理页面

>使用element UI 作为前端设计，使用Vue-router实现路由功能

>与服务器进行交互，实现数据的增删改查，并实现基本的权限管理功能

## 前端

>主要的vue使用，实现了基本的首页美化，购物车，商品等功能，与后端分离

>个别功能在后续更新会陆续添加
